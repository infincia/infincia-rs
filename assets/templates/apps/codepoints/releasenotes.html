Release notes

Codepoints 1.0.611/23/2015
  • Add support for Yosemite dark mode
  • Update to Unicode 8.0 database (NOTE: not all new Emojis in Unicode 8.0 have matching font glyphs available!)
Codepoints 1.0.57/17/2014
  • Update to Unicode 7.0 database
  • Fix bug that caused Codepoints to stay on screen when clicking other menu bar items
  • Fix bug that could potentially cause a crash while searching in rare cases
  • Significantly reduce memory usage
  • Internal restructuring and 3rd party library updates
Codepoints 1.0.48/1/2012
  • Retina icon
  • Fix bug where rows would become transparent causing problems with scrolling. May have occurred on Lion but Mountain Lion made the problem more frequent
Codepoints 1.0.35/7/2012
  • Add option to disable clearing behavior when dropdown opens, so that users can quickly go back and copy multiple related symbols without typing search terms again
Codepoints 1.0.22/3/2012
  • Fix issue with misbehaving 3rd party fonts
  • Fix issue where Codepoints would appear empty at launch
Codepoints 1.0.12/2/2012
  • Update Unicode data to version 6.1
Codepoints 1.02/2/2012
  • Initial release on Mac App Store
