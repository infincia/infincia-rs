Supported Devices

Most devices made by Novatel Wireless should work, but some carriers have modified the firmware on their branded Mi-Fi models to disable the OpenCGI API. See the FAQ for more information.

If you don't see your device listed in the table below, it means it simply hasn't been tested yet. You can test it with these easy steps (note: these links will open a new tab):

  1. Make sure your iPhone, iPad or Mac is connected to the Mi-Fi
  2. Load the Mi-Fi web interface at the default IP address: 192.168.1.1
  3. If you don't see the web interface, the IP address of your device may have been changed. Send us a support request and we will be happy to assist you
  4. Once you've verified that the web interface works, click the link below. It should load a page with a string of words like WwNetwkName, indicating that your Mi-Fi should be supported
  5. Check for OpenCGI support
