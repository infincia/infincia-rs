Case study: Hypegram for OS X

Hypegram was the first native OS X client for the Hype Machine music streaming and discovery service.

Case study: Slots Race for iOS

Slots Race is a realtime, global multi-player slot machine game for iOS. Players join a team in one of multiple themed races, and compete with both the opposing team and each other until each race ends.

Case study: Codepoints for OS X

Codepoints is a Mac utility for quickly accessing any Unicode character from anywhere on the system, written to be keyboard friendly and easy to use.

Case study: Mi-Fi Monitor for OS X

Mi-Fi Monitor for OS X is an invaluable utility for owners of Novatel Mi-Fi Hotspot devices, built as a constantly available OS X menu bar application. It was covered by The Unofficial Apple Weblog at the time of release.

Case study: Mi-Fi Monitor for iOS

Mi-Fi Monitor is an invaluable utility for owners of Novatel Mi-Fi Hotspot devices, developed in-house as our first commercial application.
